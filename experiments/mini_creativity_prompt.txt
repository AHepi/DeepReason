WHAT THIS IS
------------
This is the EXACT prompt MiniReason sent to its conjecturer LLM
(deepseek-v4-flash, temperature 1.0) to generate candidate explanations for
an open problem. Two related problems were run. Below is the wire prompt for
each on cycle 1 (before any survivors accumulate in the neighbourhood slot).

The generator returns a JSON distribution of ~6 candidate "skeletons", each a
claim + mechanism + falsifiable forbidden cases. In the real run these were
then mechanically checked (candidates that forbid nothing, or whose own
forbidden predicate does not evaluate over their text, are refuted for free),
and the survivors were ranked offline by a calibrated pairwise judge.

TO REPLICATE: paste ONE problem's block below into any model, temperature ~1.0,
and compare the candidate explanations it returns. The stance line rotates
every 5 cycles in a full run (mechanist -> skeptic -> unifier -> empiric ->
formalist -> historicist -> adversary -> minimalist); cycle 1 uses 'mechanist'.
